Aplicación móvil desarrollada con React Native + Expo Go, backend en Python (Flask) y base de datos MySQL, con autenticación mediante JWT y Refresh Token.

Requisitos
Backend

Python 3.10 o superior

pip

MySQL

Frontend

Node.js 18 o superior

npm

Expo Go (Android o iOS)

Backend (Flask)
1. Ingresar a la carpeta del backend
2. Crear el entorno virtual
python -m venv venv

3. Activar el entorno virtual
venv\Scripts\activate
4. Instalar dependencias
pip install -r requirements.txt
5. Ejecutar el servidor
python app.py

Frontend (React Native + Expo)
1. Ingresar a la carpeta del frontend
cd mobile-front/Front-app

2. Instalar dependencias
npm install

3. Levantar el proyecto con Expo
npx expo start
